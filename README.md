# banking-risk-analysis-Project
The core of this project is to transform raw operational data into strategic business intelligence. By analyzing historical trends and performance metrics, the project aims to:

Minimize Financial Risk: Identifying high-risk loan applicants through banking data patterns.

Streamline Logistics: Finding bottlenecks in the supply chain to reduce lead times and shipping costs.

Tech Stack :

SQL: Data extraction, multi-table joins, and complex aggregations (CTEs).

Python: Data preprocessing, statistical analysis, and exploratory data analysis (EDA).

Libraries: pandas, numpy for data manipulation; matplotlib, seaborn for visualization.

Environment: Jupyter Notebook
